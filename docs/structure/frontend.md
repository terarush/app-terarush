# Frontend Structure (React)

The frontend is a Vite + TypeScript application located in `/web`.

## Source Organization (`/web/src`)
- **/components/ui**: Atomic components (Shadcn/UI).
- **/components/fragments**: Composite UI components (e.g., BlogForm).
- **/lib/api**: API client and service layer.
- **/pages**: Route-level components.
- **/assets**: Global styles and static assets.
- **/hooks**: Custom React hooks.
- **/contexts**: Global state providers.
