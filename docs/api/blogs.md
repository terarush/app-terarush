# Blogs API

## Endpoints
- `GET /api/v1/blogs`: Fetch list of blogs.
- `GET /api/v1/blogs/:slug`: Fetch single blog by unique slug.
- `POST /api/v1/blogs`: Create new blog (Protected).
- `PUT /api/v1/blogs/:id`: Update existing blog (Protected).
- `DELETE /api/v1/blogs/:id`: Remove blog (Protected).
- `POST /api/v1/blogs/upload`: Image upload endpoint.
